# Skeletor - Golang HTTP Service Skeleton

```
# Clone
go get github.com/crowdriff/skeletor

# Install all required tools
cd $GOPATH/src/github.com/crowdriff/skeletor
make tools

# Install dependencies
make deps

# Run tests
make test

# Run with live-reload
make run 
# and go to http://localhost:8000

# See Makefile for more commands
```
